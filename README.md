# 🎨 HSV Color Detector

Application Python avec interface graphique (Tkinter) permettant de détecter des couleurs dans une image via des filtres HSV.

## ✨ Fonctionnalités

- Détection Rouge / Vert / Bleu
- Ajustement précis des plages HSV
- Pipette interactive (clic sur image)
- Affichage des zones détectées
- Statistiques (%) de présence des couleurs

## 🖥️ Technologies

- Python
- Tkinter
- OpenCV
- NumPy
- Pillow
